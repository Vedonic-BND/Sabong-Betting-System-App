CALLBACK OWNERSHIP MODEL
────────────────────────

RunnerViewModel (Owns Runner-Specific Callbacks)
    ├─ onConnected
    ├─ onDisconnected
    ├─ onTellerCashUpdated (from cash-status channel)
    ├─ onCashRequested
    ├─ onRunnerAccepted
    └─ onRunnerAssignedByOwner ← THIS IS THE TARGET ✓

ReverbViewModel (Owns Fight/Betting Callbacks)
    ├─ onFightUpdated
    ├─ onBetPlaced
    ├─ onWinnerDeclared
    ├─ onTellerCashUpdated (from fights channel - separate)
    ├─ onRunnerAccepted (for teller filtering)
    └─ onRunnerDeclined

Global Singleton (ReverbManager)
    ├─ Lifetime: App startup to app close
    ├─ Multiple ViewModels can read from it
    ├─ Each ViewModel manages only its callbacks
    └─ No ViewModel should call disconnect() ✓
